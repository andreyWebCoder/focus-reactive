-------------------Верстка проекта focus-reactive----------------------------
1) Развертывание проекта внедрение в проект шаблонизатора nunjucks, настройка scss
	переменных, подключение шрифтов, вигрузка картинок и иконок;
2) Структура проекта:
		2.1) Папка focus-reactive -- результат c файлами (css, js, fonts, ims, html);
		2.2) Папка gulp -- файли с настройки сборщика gulp;
		2.3) Папка с иходниками src коториє обрабативает gulp:
			2.3.1) папка fonts исходники для шрифтов в формате .tft;
			2.3.2) папка html разбитые файли на компоненти с разшиернием .njk;
			2.3.3) папка scss файли препроцесора стилей которие компилируют стили;
			2.3.4) файли с разшиернием .njk в корне src ето страници в которие инклудяться компоненти и 
			компилируеться html;
		2.4) gulpfile.js сценарии для сборщика gulp;
		2.5) package.json пакети плагинов gulp;
		2.6) readme.txt Описание проекта;
