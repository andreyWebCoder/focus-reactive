------------------- Верстка проекта focus-reactive----------------------------

------------------- Структура проекта----------------------------

1) Структура проекта:
		1.1) Папка focus-reactive -- результат c файлами (css, js, fonts, ims, html);
		1.2) Папка gulp -- файли с настройки сборщика gulp;
		1.3) Папка с иходниками src коториє обрабативает gulp:
			1.3.1) папка fonts исходники для шрифтов в формате .tft;
			1.3.2) папка html разбитые файли на компоненти с разшиернием .njk;
			1.3.3) папка scss файли препроцесора стилей которие компилируют стили. vendors  файл для подключения и 
			компиляции сторонних библиотек (отключен в сценариях gulp), файл style для подключения кастомних стилей и
			их компиляции;
			1.3.4) файли с разшиернием .njk в корне src ето страници в которие инклудяться компоненти и 
			компилируеться html;
		1.4) gulpfile.js сценарии для сборщика gulp;
		1.5) package.json пакети плагинов gulp;
		1.6) readme.txt Описание проекта;

------------------- Краткое описание ----------------------------

1) Развертывание проекта внедрение в проект шаблонизатора nunjucks, настройка scss
	переменных, подключение шрифтов, вигрузка картинок и иконок;
2) В Figma применил 2 плагина Inspect Styles (инспектирует минимальные стили css), Export Original Imajes( 
	вигружает изображения в оригинале);
3) Розбивка дизайна по компонентно находиться в папке src/html;
4) В проекте в scss применяеться миксин которий адаптирует шрифти заголовков в зависимости от вюпорта 
если екран менее container и до 768px ширини вюпорта (src/scss/mixins/_resources.scss). Также этот миксин 
применялся и для отзывчивости некоторих ширин и отступов;
5) В стилях заложени переменные scss (src/scss/vars/_vars.scss);
6) На странице создана оболочка wrapper которая прибивает футер к низу c с помощью main;
7) Создан ограничивающий контейнер что бы на екранах более 1440px контен не розпливался;
8) Header -- лого сделано ссилкой для возврата на главную страницу, пунктам меню навешен ховер также заложено 
состояние active, так же навигация на мобильних устройствах прячеться в меню бургер;
9) Первий екран сделан фуллскрином ( hero section) Фон сделан картинкой для удобства загрузки из админки;
10) Второй екран так же фуллскрин + анимирован с помощью библиотеки GSAP (https://gsap.com/) -- src/js/files/animate.js;
11) Трейтий екран анимации айтемов, cделан так что 1 айтем будет всегда открыт (work section) дает понимае юзеру что атемы
розкриваемы. Айтеми project зазбиты на десктоп версию и моб версию в соответствии отображаються.
12) Всем кнопкам и ссылкам добавлен минимальный ховер и курсор поинтер Для понимания что они кликабельни;
13) Изображения оптимизированы и добавлены в формат webp который подставлен в структуру страницы и проверяеться на поддержку ; 
14) В папке img находиться файл favicon icon.svg
15)Использованы инструменты ИИ (ChatGPT) воосновном для поиска иформации для внедрение шаблонизатора nunjucks также настройку новой версии
	scss. Правки некоторих частей (минимально) js кода.

------------------- Работа со сборкой gulp ----------------------------
В терминале:
$ npm i
$ npm run dev -- Запуск шаблона в режиме разработки
$ npm run build -- Компилит все необходимиє файлы min, шрифти, webp(Конечниє файли)
$ npm run zip -- Создает архи с исходниками
